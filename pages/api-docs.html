API Documentation

REST API for the QuantSolve Portfolio Engine. Base URL: http://localhost:3000

GET /market/quotes

Fetch live NSE stock prices for one or more symbols.

Parameters
ParamTypeDescription
symbolsstringComma-separated NSE ticker symbols (e.g. RELIANCE,TCS,INFY). Max 6.
Example response
{
  "ok": true,
  "source": "live",
  "data": {
    "RELIANCE": { "symbol": "RELIANCE", "price": 2948.50, "name": "Reliance Industries", "currency": "INR" },
    "TCS":      { "symbol": "TCS",      "price": 3812.75, "name": "Tata Consultancy",    "currency": "INR" }
  }
}

POST /solve/v2

Run the integer-linear solver on a budget equation with per-variable constraints.

Request body
FieldTypeDescription
equationstringLinear equation. E.g. 2948a + 3812b = 500000
constraintsobjectPer-variable bounds. E.g. {"a":{"max":50},"b":{"max":30}}
curl
curl -X POST http://localhost:3000/solve/v2 \
  -H "Content-Type: application/json" \
  -d '{"equation":"2948a + 3812b = 500000","constraints":{"a":{"max":50},"b":{"max":30}}}'
JavaScript (fetch)
const res = await fetch('/solve/v2', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    equation: '2948a + 3812b = 500000',
    constraints: { a: { max: 50 }, b: { max: 30 } },
  }),
});
const { ok, data } = await res.json();
// data = [{ a: 42, b: 24 }, ...]
