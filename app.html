API Documentation

REST API for the QuantSolve Portfolio Engine. Base URL: http://localhost:3000

GET /market/quotes

Fetch live (or demo) stock prices for one or more symbols.

Parameters
ParamTypeDescription
symbolsstringComma-separated ticker symbols (e.g. AAPL,MSFT,NVDA). Max 6.
Example response
{
  "ok": true,
  "source": "static",
  "data": {
    "AAPL": { "symbol": "AAPL", "price": 196.89, "name": "Apple Inc." },
    "MSFT": { "symbol": "MSFT", "price": 421.55, "name": "Microsoft Corporation" }
  }
}

POST /solve/v2

Run the integer-linear solver on a budget equation with per-variable constraints.

Request body
FieldTypeDescription
equationstringLinear equation using single-letter variable names. E.g. 19689a + 42155b + 1k = 500000
constraintsobjectPer-variable bounds. E.g. {"a":{"max":25},"k":{"max":5000}}
Code examples
curl 
curl -X POST http://localhost:3000/solve/v2 \
  -H "Content-Type: application/json" \
  -d '{"equation":"19689a + 42155b + 1k = 500000","constraints":{"a":{"max":25},"b":{"max":11},"k":{"max":5000}}}'
JavaScript (fetch)
const res = await fetch('/solve/v2', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    equation: '19689a + 42155b + 1k = 500000',
    constraints: { a: { max: 25 }, b: { max: 11 }, k: { max: 5000 } },
  }),
});
const { ok, data } = await res.json();
// data = [{ a: 21, b: 2, k: 2221 }, ...]
